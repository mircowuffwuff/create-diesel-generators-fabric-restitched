<h1 align="center">How to contribute</h1>

there are many different ways to set up a working development environment.<br>if you dont know where to start, this is how i set mine up on windows!
<br>
<br>
prerequisities: <a href="https://git-scm.com/downloads">git</a>, <!--<a href="https://cli.github.com/">GitHub CLI</a>, --><a href="https://www.oracle.com/de/java/technologies/downloads/#java17-windows">JDK 17</a>, <a href="https://www.jetbrains.com/idea/download/download-thanks.html?platform=windows&code=IIC">IntelliJ IDEA</a>, a GitHub account
<details>
<summary><h3 align="left">fork</h3></summary>
<p align="left">
log into GitHub and fork the repo by hitting Fork, followed by the green Create fork button <!--![hjgkfdslhguishgl](https://github.com/user-attachments/assets/2fb95036-b2f4-471c-94d9-2b180b8f0487)-->
<br><img alt="ueisojhgoiesg" src="https://github.com/user-attachments/assets/e0815c36-be6d-4235-bf2a-5ed8fcb9cb66" />
</p>
</details>
<details>
<summary><h3 align="left">clone</h3></summary>
<p align="left">
navigate to your fork on github. hit the green code button, followed by the copy button
<br><img alt="gdskgdsk" src="https://github.com/user-attachments/assets/69218e10-3138-4651-a3e4-7173e99cc34d" />
<br>open a command line, navigate to your projects folder, input `git clone `, paste the url you copied earlier and hit enter
<br><img width="412" alt="sadfsagdggsd" src="https://github.com/user-attachments/assets/62aea832-9723-4380-8eca-629473c10896" />
</p>
</details>
<details>
<summary><h3 align="left">open</h3></summary>
<p align="left">
open up IntelliJ IDEA, hit open, navigate to your Projects folder, select the freshly cloned repo folder, open and finally hit trust
<br><img width="320" alt="dagsgdgsgsd" src="https://github.com/user-attachments/assets/2bbe1da5-7d0b-4cf2-9c98-1f21866bb887" />
<br>wait for the Gradle Project to finish importing
<br><img width="309" alt="vgcgfgfsdhfdh" src="https://github.com/user-attachments/assets/d3b17fac-62c7-46de-be60-41504651b338" />
</p>
</details>
<details>
<summary><h3 align="left">build</h3></summary>
<p align="left">
once you changed something, hit the terminal icon, paste `.\gradlew clean build` and hit enter. your built mod jar will be at `Projects\create-diesel-generators-fabric-restitched\build\lib\createdieselgenerators-fabric-restitched-xxxxx.jar`
<br><img width="447" alt="fsgrghsgrg" src="https://github.com/user-attachments/assets/763b5a64-27e3-4dd1-a57b-59a3a71bb090" />
</p>
</details>
<details>
<summary><h3 align="left">test</h3></summary>
<p align="left">
insert that jar into a testing environment consisting of fabric 1.20.1, fabric-api, create-fabric and the newly built jar and test rigorously
<br><img width="471" alt="pgfhfhfdh" src="https://github.com/user-attachments/assets/49abf041-6240-4a17-97e9-e3d41bdce8de" />
</p>
</details>
<details>
<summary><h3 align="left">commit</h3></summary>
<p align="left">
once testing is successful, hit the commit button, choose the files you want to commit, describe your commit briefly and hit commit and push your changes
<br><img width="471" alt="0fjw329jf3j" src="https://github.com/user-attachments/assets/ffc16d9d-e164-4c7e-b923-5310c34e005d" />
<br>if prompted, enter your email and continue. IDEA might warn about existing warnings, just hit Commit and Push anyways. if prompted to log in to GitHub, do so and follow the on screen steps. you will only have to do these once.
</p>
</details>
<details>
<summary><h3 align="left">contribute</h3></summary>
<p align="left">
once you are sure, that your fork, with all of its commits, is stable and should be part of the next Create: Diesel Generators [Fabric Restitched] update, pull up your fork's GitHub page, hit Contribute followed by the green Open pull request button and follow the steps on screen
<br><img width="323" alt="vhbiengveg" src="https://github.com/user-attachments/assets/912bbe6a-5fee-4469-bf87-eaf1c58b2693" />
</p>
</details>
